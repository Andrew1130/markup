h3 제목

하나의 문서 내부에 필요한 여러 문단, 문장을 작성하기 위해 사용하는 요소

내용을 작성하는 형태에서, 띄어쓰기 처리는 한칸만 허용, 내용상의 줄바꿈처리는 허용하지 않고 띄어쓰기로 처리된다.

하나의 내용을 가지고 있는 문단에서,
줄바꿈을 처리하는 것은 전혀 다른 의미이므로,
하나의 문단에서 줄바꿈을 처리하기 위해서는
현 위치를 기준 줄바꿈 처리하는 요소를 넣는다.

문단을 의미하는 p요소와는 다르게,        사용하는 형태 모두 
그대로 유지하는 요소이지만, 불편함이 많아 사용률이 적다. 
html의 요소 형태 그대로 표기할 때 주로 사용한다.
특수문자를 사용하는 경우에는 entity코드를 사용하자.
